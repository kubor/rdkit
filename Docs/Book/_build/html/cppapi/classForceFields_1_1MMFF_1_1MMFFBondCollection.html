static MMFFBondCollection* ForceFields::MMFF::MMFFBondCollection::getMMFFBond ( const std::string &  mmffBond = "")
static

gets a pointer to the singleton MMFFBondCollection

Parameters
mmffBond(optional) a string with parameter data. See below for more information about this argument
Returns
a pointer to the singleton MMFFBondCollection

Notes:

  • do not delete the pointer returned here
  • if the singleton MMFFBondCollection has already been instantiated and mmffBond is empty, the singleton will be returned.
  • if mmffBond is empty and the singleton MMFFBondCollection has not yet been instantiated, the default parameters (from Params.cpp) will be used.
  • if mmffBond is supplied, a new singleton will be instantiated. The current instantiation (if there is one) will be deleted.
