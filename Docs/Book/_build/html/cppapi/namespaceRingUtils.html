For each ring in bring compute and strore the ring that are fused (share atleast one bond with it).

Useful both for the keulization stuff and aromaticity perception.

Parameters
bringslist of rings - each ring is specified as a list of bond IDs
neighMapan STL map into which the results are stored. Each entry in the map is indexed by the ring ID and the conents are the list rings (rather their IDs) that are fused with this ring
maxSizeif this is >0, rings that are larger than the threshold will not be considered as candidates to be neighbors
